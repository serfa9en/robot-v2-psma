# robot-v2-psma

Веб-приложение для медицинского робота Promobot V4 от ПГМУ

## Описание

Робот имеет разные мед устройства для измерения показателей здоровья:
* тонометр
* датчик температуры
* глюкометр
* пульсоксиметр

Прохождение опросника от специалиста (в формате опроса):
* Кардиолог
* Невролог
* Гастроэнтеролог
* Гинеколог
* Уролог
* Колопроктолог
* Офтальмолог
* Оториноларинголог
* Эндокринолог

Прохождение опросника для расчета риска развития заболевания:
* Сахарный диабет
* Рефлюксная болезнь
* Депрессия
* Простатит
* Рак молочной железы
* Рак желудка
* Рак легких
* Рак пищевода
* Рак предстательной железы, рак мочевого пузыря
* Меланома
* Рак кожи
* Рак органов полости рта и глотки
* Рак гортани
* Рак щитовидной железы
* Рак кишечника

Распечатать полный отчет после прохождения разных обследований
